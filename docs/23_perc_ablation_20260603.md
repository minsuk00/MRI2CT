# U-Net perceptual-loss ablation

**Source HTML:** _html/perc_ablation_20260603.html
**Date:** 2026-06-03
**TL;DR:** Earliest snapshot of the perc_ablation series, at the ep400/200k-step checkpoint. The clean perceptual contrast (perc 0.5 vs 0, dice 0.1 both) shows perceptual marginally helps body MAE (macro 31.9 vs 32.3 HU, significant Wilcoxon p=2.0e-03) with Bone Dice essentially tied (0.822 vs 0.821). The perceptual,no-dice run isolates dice: dropping dice tanks Hard Bone Dice (0.753 vs 0.822) and all-class Hard Dice (0.646 vs 0.730).

Earliest of the perc_ablation series (docs/24 dated 2026-06-09 at ep200/100k steps, docs/25 dated 2026-06-17 broadening to anatomix-injection site). Confound caveat: these runs vary on two axes (perceptual_w and dice_w). The clean perceptual contrast is no-perc (perc 0, dice 0.1) vs perceptual (perc 0.5, dice 0.1), identical dice, differing only in perceptual loss. The perceptual,no-dice run isolates the dice contribution. Do not read the Hard-Dice gap to the no-dice run as a perceptual effect.
